# Panduan Kontribusi 🤝

Terima kasih sudah mau berkontribusi di **Kitab Oracle Cloud (VPS Gratis Selamanya)**! Repo ini ada untuk membantu developer Indonesia menemukan *script boilerplate* dan *tools* Supabase terbaik panduan Oracle Cloud Free Tier terbaik.

## Cara Menambahkan Tools/Template

1. **Fork** repositori ini.
2. Tambahkan entri Anda ke kategori yang sesuai dalam format tabel:
   ```markdown
   | [nama-repo](https://github.com/user/repo) | Deskripsi singkat berbahasa Indonesia. |
   ```
3. Pastikan proyek memenuhi **Kriteria Kurasi**:
   - Berjalan atau memanfaatkan Oracle Cloud Always Free (permanen).
   - Automasi provisioning, self-hosting, atau IaC.
   - *Open-source* dan idealnya masih aktif di-*maintain*.
   - Bermanfaat untuk konteks developer Indonesia.
4. Buat **Pull Request** dengan judul jelas, contoh: `add: [nama-tool] ke kategori Auto-Provisioning ARM`.

## Aturan Deskripsi
- Gunakan Bahasa Indonesia yang jelas dan ringkas.
- Jelaskan **manfaat**-nya, bukan sekadar menyalin judul repo.
- Cantumkan jika ada catatan penting (misal: butuh paket berbayar).

Terima kasih! 🚀
